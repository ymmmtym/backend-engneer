    Tmux

    Requirements

    • tmux > 3.0

    Commands

    tmux new -s ${session}

tmux kill-session -t ${session}
tmux kill-server

    Prefix

    Prefix is Ctrl+b command. Controll tmux, type prefix and any following key.

    Session

    Key Action
    : prompt
    ? show list of key bind
    s select session
    d detatche session
    $ rename current session

    Window

    Key Action
    c create new window
    w show window list
    n move next window
    p move previous window

    Pane

    Key Action
    % split pane horizontally
    " split pane vertically
    o move next pane
    ; move previous pane
    Space change layout

    Config

    put file ~/.tmux.conf or /etc/tmux.conf
